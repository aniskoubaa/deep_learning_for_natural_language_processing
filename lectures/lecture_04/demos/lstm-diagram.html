LSTM: Long Short-Term Memory

The repeating module in an LSTM contains four interacting layers (gates), controlling what information flows through the cell state.

Cell body (A)
Sigmoid gate (σ)
Tanh layer
Pointwise operation
Vector flow
Vector state
1. Forget Gate
$$f_t = \sigma\big(W_f \cdot [h_{t-1},\, x_t] + b_f\big)$$
Decides what to throw away from the cell state. Outputs 0 (forget) to 1 (keep) for each dimension.
2. Input Gate + Candidate
$$i_t = \sigma\big(W_i \cdot [h_{t-1},\, x_t] + b_i\big) \qquad \tilde{C}_t = \tanh\big(W_C \cdot [h_{t-1},\, x_t] + b_C\big)$$
Decides what new information to store. The input gate scales the candidate values.
3. Cell State Update
$$C_t = f_t \odot C_{t-1} \;+\; i_t \odot \tilde{C}_t$$
The cell state is updated: forget old (multiply by f_t) then add new (scaled candidate).
4. Output Gate
$$o_t = \sigma\big(W_o \cdot [h_{t-1},\, x_t] + b_o\big) \qquad h_t = o_t \odot \tanh(C_t)$$
Decides what parts of the cell state to output as the hidden state.
